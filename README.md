# 📊 Sales Data Analytics Dashboard

An interactive web-based dashboard for analyzing sales data using CSV uploads.  
Built with JavaScript, this tool provides real-time insights through charts, metrics, and filters.

---

## 🚀 Features

- 📂 Upload CSV file for instant analysis  
- 💰 View key metrics (Total Sales, Profit, Orders)  
- 📊 Bar Chart (Sales by Category)  
- 📈 Line Chart (Sales Over Time)  
- 🥧 Pie Chart (Category Distribution)  
- 🔍 Filter data by category  
- 📋 Dynamic data table  
- 🎨 Clean and responsive UI  

---

## 🛠️ Tech Stack

- HTML  
- CSS  
- JavaScript  
- Chart.js (for data visualization)  
- PapaParse (for CSV parsing)
